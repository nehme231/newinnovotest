# libs

This directory contains shared libraries used across the project.

Structure:
- libs/animations — animation assets or helper modules
- libs/icons — shared icon components or SVGs
- libs/ui — shared UI components (buttons, inputs, layout helpers)
- libs/utils — small utility functions/helpers

Usage:
- Add components or modules inside the appropriate subfolder.
- Export commonly-used helpers from index files as needed.
- Keep each library focused and well-documented.
